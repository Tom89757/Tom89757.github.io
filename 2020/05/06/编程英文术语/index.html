以下内容按字母顺序排列

A

ASLR：地址空间布局随机化(Address-Space Layout Randomization)。从用ASLR，每次运行时程序的不同部分，都会被加载到内存的不同区域。

B

BIOS：基本输入输出系统（Basic Input/Output System），亦称为ROM BIOS、System BIOS、PC BIOS，是在通电引导阶段运行硬件初始化，以及为操作系统提供运行时服务的固件。BIOS预安装在个人电脑的主板上，是个人电脑启动时加载的第一个软件。

bus：总线，一组并行的导线，能携带地址、数据和控制信号。

C

CISC和RISC：复杂指令集计算机和精简指令集计算机。

D

DIMM：双列直插内存模块（Dual Inline Memory Module），使用240个引脚，以64位为块传送数据到内存控制器和从内存控制器传出数据。

DMA：直接内存访问（Direct Memory Access），设备自己执行读或者写总线事务而不需要CPU干涉。如CPU从磁盘读数据过程中磁盘控制器确定目标扇区并将信息传送给主存。

F

flash memory：闪存，一类非易失性存储器，基于EEPROM，目前基于闪存的固态硬盘SSD大受欢迎。

firmware：固件，是一种嵌入在硬件设备中的软件，在系统通电时运行。通常它是位于特殊应用集成电路（ASIC）或可编程逻辑器件（PLD）之中的闪存或EEPROM或PROM里，有的可以让用户更新。可以应用在非常广泛的电子产品中，从遥控器、计算器到电脑中的键盘、硬盘，甚至工业机器人中都可见到它的身影。

G

GCC：GNU编译器套装（GNU Compiler Collection），GNU计划的关键部分，也是GNU工具链的主要组成部分之一。GCC（特别是其中的C语言编译器）也常被认为是跨平台编译器的事实标准。

I

ISA：指令集架构（Instruction Set Architecture），用以定义机器机程序的格式和行为，定义了处理器状态、指令的格式，以及每条指令对状态的影响。

R

RAM：随机访问存储器，Random-Access Memory，分为静态的和动态的，即SRAM和DRAM，静态比动态更快、更贵。

ROM：只读存储器，Read-Only Memory，其中有的类型可读也可写，其通过能够被重编程的次数以及重编程机制进行区分。

S

SIMD：单指令多数据（Single instruction Multiple Data）。通过SIMD并行可以同时对一组数据执行相同的操作从而实现空间上的并行。

SSD：一种基于闪存的存储技术，一个SSD封装由一个或多个闪存芯片和闪存翻译层组成，闪存芯片与传统磁盘中由多个盘片封装成的机械驱动器对应，硬件/固件构成的翻译层则与磁盘控制器对应。

SSE指令：流SIMD（Streaming SIMD Extensions）。SSE历经几代，已经更新为AVX（高级向量扩展，advanced vector extension），可以通过一条指令产生对多个数据值的计算，这些值存储在一组特殊的向量寄存器中。

straight-line code：无分支结构（如while; do-while; for）的代码

system call：系统调用，应用程序通过系统调用指令将处理器的控制权交由操作系统内核，内核执行被请求的操作后将结果返回给应用程序。

P

prompt: 提示，提示符

U

USB：通用串行总线（Universal Serial Bus），USB控制器是一个连接到USB总线的设备的中转机构，用以连接各种外围I/O设备。
