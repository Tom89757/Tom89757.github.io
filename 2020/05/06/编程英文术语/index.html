编程英文术语

  1. D
  2. G
  3. I
  4. S
  5. P

以下内容按字母顺序排列

D

DRAM: 动态随机访问存储器mark

G

GCC：GNU Compiler Collection的缩写，GNU编译器套装，以GPL及LGPL许可证所发行的自由软件，也是GNU计划的关键部分，也是GNU工具链的主要组成部分之一。GCC（特别是其中的C语言编译器）也常被认为是跨平台编译器的事实标准。

I

ISA：Instruction Set Architecture的缩写，指令集架构，用以定义机器机程序的格式和行为，定义了处理器状态、指令的格式，以及每条指令对状态的影响。

S

SRAM: 静态随机访问存储器mark

SIMD：Single instruction Multiple Data。通过SIMD并行即单指令多数据并行可以同时对一组数是执行相同的操作从而实现空间上的并行。

system call：系统调用，应用程序通过系统调用指令将处理器的控制权交由操作系统内核，内核执行被请求的操作后将结果返回给应用程序。

P

prompt: 提示，提示符
