本文记录一下在开发前端项目时的知识点和所遇到的问题：
<!--more-->

1.报错`Support for the experimental syntax 'jsx' isn't currently enabled`。
问题：缺少`.babelrc`文件
解决方案：在项目根目录创建一个`.babelrc`文件，并添加如下内容
```javascript
{
  "presets": ["@babel/preset-env", "@babel/preset-react"]
}
```
